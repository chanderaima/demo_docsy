To create a new pipeline, follow the next steps:

  1. Create a configuration file config.yaml:

    apiVersion: tekton.dev/v1beta1
        
    kind: Task
        
    metadata:
        
      name: hello
        
    spec:
        
      steps:
        
        - name: echo
        
          image: alpine
        
          script: |
        
            #!/bin/sh
        
            echo "Hello World"

  2. Apply the file to your cluster kubectl apply config.yaml
